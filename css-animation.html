BRINGING A CHARACTER TO LIFE

This was one of my first projects as a Multimedia Design student. Using Adobe Illustrator and CSS animations, the task was to design your own character, build a storyline around it and breathe life into the character by drawing a sprite sheet and animating one scene from the story.



It was a fun exercise which opened my eyes to the many possibilities and applications of CSS animations in web design.

I didn’t have an exact vision as to what I wanted the character to look like, except I was going for flat design. I was experimenting with basic shapes and the pen tool in Illustrator, and just from playing around with the shapes and combining them, this cute, little Grim Reaper began to take form.

My inspiration for the universe was the game ‘Year Walk’. For those not familiar with it, Year Walk is a Swedish adventure game developed for iOS devices in 2013. The game is loosely based on an ancient Swedish tradition called "Årsgång". It has en eerie feel to it which I thought could go well with the little Grim Reaper. This is the style tile which I designed to provide the style and atmosphere for the scene.

The background I designed in Adobe Illustrator as well. I designed the shapes myself and applied glow and texture filters.

In order to create the animation, I had to draw the character in several poses in order to design the sprite sheet. It was challenging, but fun, to draw the different facial expressions and see the character come to life. Using HTML and CSS, I used CSS keyframes to run an animation of the sprite sheet.
